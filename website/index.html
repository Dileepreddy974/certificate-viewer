Key Features

📋

Full Certificate Chain

Retrieves complete certificate chain from any HTTPS endpoint

🔍

Detailed Information

Displays Subject, Issuer, CN, validity period, SHA-256 fingerprint, and SANs

✅

Certificate Validation

Performs expiration check and hostname verification with wildcard support

⚡

Fast & Lightweight

Built with native Java APIs, no external dependencies required

💻

Multiple Modes

CLI mode for direct usage, interactive mode for prompt-based input

🔒

Security Focused

Uses javax.net.ssl and java.security.cert for secure operations
